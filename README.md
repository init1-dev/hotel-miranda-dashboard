# hotel-miranda-dashboard

Dashboard app made using React with styled-components (only desktop, no responsive)

# Additional packages:

- @fullcalendar
- date-fns
- react-icons
- sweetalert2
- swiper

# Usage instructions (dev):

- clone repo
- npm i
- checkout to dev branch
- npm run dev
