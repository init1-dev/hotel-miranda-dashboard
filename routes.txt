# Bookings:
    - Listar todos los bookings: GET /bookings
    - Mostrar un booking específico: GET /bookings/:id
    - Crear un nuevo booking: POST /bookings
    - Actualizar un booking existente: PUT /bookings/:id
    - Eliminar un booking existente: DELETE /bookings/:id

# Rooms:
    - Listar todas las rooms: GET /rooms
    - Mostrar una room específica: GET /rooms/:id
    - Crear una nueva room: POST /rooms
    - Actualizar una room existente: PUT /rooms/:id
    - Eliminar una room existente: DELETE /rooms/:id
